# Doctor Assistant

A Python application for document summarization and medical assistance.

## Project Structure

- **backend/**: Backend logic including document summarization
  - `main.py`: Main backend application
  - `document_summarization.py`: Document summarization functionality
  
- **frontend/**: Frontend interface
  - `streamlit.py`: Streamlit-based web interface

## Installation

1. Clone this repository
2. Install required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the Streamlit application:
```bash
streamlit run frontend/streamlit.py
```

## Requirements

See `requirements.txt` for dependencies.
